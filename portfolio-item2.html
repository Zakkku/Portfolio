Buff Boiislanding page

Figma, HTML, CSS

March, 2022

For this project I was given a design created in Figma (a software similar to Adobe Illustrator) and had to code out the website from scratch. Some details were quite challenging to get looking like the desgin but I worked hard and learned how to do new things by checking documentation, Stack Overflow, etc.

This was the third or fourth project I did using the mobile first approach. The first static web pages I made in the beginning of my journey were mostly desktop-only just because I needed to focus on the basics of web development without making things too complicated.

After my hiatus from we development to manage my own Shopify store I rebuilt some projects from scratch just to brush up. This was one of the projects I redid, in May of 2023.
