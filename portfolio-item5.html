October, 2023

Although this project is just vanilla JavaScript, HTML, and CSS, I feel like I worked harder on this than some React apps I've been building. I learned more about data structures and JavaScript functions while making this project than I did from most tutorials. I think it's actually the longest script file I've ever written at the time of writing this.

I made this quiz when some of my friends were interested in getting into web development but weren't sure if they can do it or not. After spending around an hour searching on the internet I realized almost all the quizzes out there are made by coding bootcamps, trying to funnel you into their program and get $10,000+ from you.

If you know anyone who is considering web develpment as a career please share this with them!
